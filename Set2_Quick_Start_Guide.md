## 🔧 トラブルシューティング

### 問題: 問題データが読み込まれない
```
症状: ホーム表示後、大問1クリック → 白い画面

原因:
1. GAS がオフライン
2. 問題データ JSON が不正

対応:
1. GAS URL で "GAS is running" メッセージ確認
2. ブラウザコンソール (F12) でエラー確認
   - network タブで Status 確認
   - console タブで error メッセージ確認
```

### 問題: 音声ファイルが再生されない
```
症状: リスニング再生ボタン → 音が出ない

原因:
1. GitHub リポジトリが削除・非公開
2. MP3 ファイルが存在しない
3. URL が不正

対応:
1. GitHub で確認:
   https://github.com/kj-eiken/eiken-audio
   └─ Set2_P1_No1.mp3 等が存在

2. HTML audio タグの src を確認
   <audio src="...mp3"></audio>

3. ブラウザコンソールで CORS エラー確認
```

### 問題: メール送信エラー
```
症状: 「送信」ボタン → エラーメッセージ

原因:
1. Gmail API が有効になっていない
2. GAS がメール権限を持っていない
3. MAIL_TO アドレスが不正

対応:
1. GAS でテスト実行:
   function testMail() {
     GmailApp.sendEmail('online-eigojuku@toshin.com', 'テスト', '本文');
   }

2. GAS の権限確認（デプロイ時）

3. スプレッドシート Sheet1 に 00000001 / テスト が存在確認
```
