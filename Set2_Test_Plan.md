## 📝 テスト実行方法

### 事前準備
1. **テスト用 Google アカウント** を作成
2. **スプレッドシート** にアクセス権確認
3. **メール受信用アカウント** (online-eigojuku@toshin.com) を準備
4. **各ブラウザのインストール** 確認

### テスト環境設定
```
1. GAS URL にアクセス
2. ログイン ID: 00000001 / 名前: テスト
3. 各テストケース実行
4. 結果を本シートに記録
```

### 結果記録方法
- **実結果列**: テスト実行後、観察結果を記入
- **判定列**: ✅ (PASS) / ❌ (FAIL) / ⚠️ (要確認)
- **備考**: 失敗時は詳細・スクリーンショット添付
