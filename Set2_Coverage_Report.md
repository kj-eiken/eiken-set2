## 🔧 技術詳細 ・ 確認ポイント

### バックエンド (GAS)

**関数呼び出しフロー**:
```
doPost(params)
  ↓
  - params.action = 'getQuestions' → handleGetQuestions()
    ├─ QD (大問1〜4のデータ)
    ├─ LIS_Q (リスニング全30問)
    └─ Q5_MODEL / Q6_MODEL (英作文モデル)

  - params.action = 'send' → handleSend(params)
    ├─ 解答データをメール送信
    └─ スプレッドシート「結果集計_Set2」に記録
```

**リスニング音声URL形式**:
```javascript
// GitHub raw URL
audioSrc: "https://raw.githubusercontent.com/kj-eiken/eiken-audio/main/Set2_P1_No1.mp3"
// P = Part, No = 問番号
```

**採点関数** (フロントエンド側):
- `checkSection('q1')` → 大問1全15問の採点
- `checkSection('q2')` → 大問2全5問の採点
- `submitAnswers()` → 全問の送信・メール・スプレッドシート記録
